# 新年度 端末配布・初期設定チェックリスト
（学校向け / Google環境想定）

## 想定
- 対象：学校・学習塾の端末配布担当（先生・ICT担当）
- 環境：Google Workspace / Google Classroom
- 端末：Chromebook または Windows + Chrome

---

## 目次

### 0. 配布前準備
- 配布方式・動線の確認
- 端末台帳の準備
- アカウント一覧と配布対象の一致確認
- Wi-Fi情報の整理
- 当日の役割分担

### 1. アカウント・ログイン確認
- 初期パスワードと変更ルール
- ログイン制限・多要素認証の有無

### 2. 端末初回起動
- 電源ON〜Wi-Fi接続
- Googleアカウントでログイン
- 同期・規約画面での注意点

### 3. Google Classroom
- クラス参加方法
- 表示確認
- 課題提出テスト

### 4. Chrome / Drive 基本設定
- ブックマーク設定
- ファイル保存・共有ルール

### 5. セキュリティ・運用ルール
- パスワード管理
- 端末破損・紛失時の対応

### 6. よくあるトラブル
- ログインできない
- Wi-Fiにつながらない
- Classroomに入れない

### 7. 当日用チェック
- 配布手順の流れ
- 1台あたりの所要時間

---

※ 各項目は  
「やること / 確認ポイント / 失敗時の対処」  
の形式で後日記載予定
