# Windowsキッティング手順書（チェックリスト）

## 概要
社内で新規Windows PCを配布する際のキッティング作業を想定し、  
設定漏れや属人化を防ぐために作成したチェックリスト形式の手順書です。

## 想定環境
- OS：Windows 10 / Windows 11
- 想定規模：中小企業（情シス1〜数名体制）
- 管理形態：手動設定（Intune等のMDM未導入環境を主に想定）
- アカウント：Microsoft 365（個人アカウント／組織アカウント）

## 目的
- 作業手順の標準化による設定漏れ防止
- 担当者が変わっても再現できる状態を作る
- 将来的な自動化・MDM導入の土台とする

## 事前準備（配布前）
- [ ] 端末情報（メーカー／型番／シリアル）の確認
- [ ] 配布先ユーザー（氏名・部署・権限）の確認
- [ ] PC命名ルールの確認
- [ ] 管理者アカウントの準備
- [ ] 必要な業務アプリ・ライセンスの確認

## キッティング手順

### 1. 初期設定
- [ ] PC名の設定
- [ ] Windows Updateの適用
- [ ] タイムゾーン／言語設定の確認
- [ ] ローカル管理者の設定方針確認

### 2. セキュリティ設定
- [ ] BitLockerの有効化
- [ ] BitLocker回復キーの保存先確認（台帳／管理者のみ閲覧可）
- [ ] Microsoft Defenderの有効化・更新
- [ ] 画面ロック設定（スリープ・パスワード）

### 3. アカウント・アプリ
- [ ] Microsoftアカウント／業務アカウントでサインイン
- [ ] Microsoft 365（Outlook / Teams 等）の設定
- [ ] ブラウザ設定（既定・ブックマーク）
- [ ] 業務アプリのインストール

### 4. ネットワーク・周辺機器
- [ ] 社内Wi-Fi接続確認
- [ ] VPN設定（必要な場合）
- [ ] プリンタ・周辺機器設定
- [ ] カメラ／マイク動作確認

### 5. 受け渡し前最終確認
- [ ] 再起動後の正常動作確認
- [ ] 業務利用に問題がないか確認
- [ ] 資産管理台帳への登録
- [ ] 利用者向け簡易案内の作成・共有
